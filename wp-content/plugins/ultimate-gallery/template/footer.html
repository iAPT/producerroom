USE

Click edit to set the video gallery. Add, edit, or delete the videos you want to display the gallery. You can see here the options you have for video:

  • Youtube videos or video flv. You can put a youtube url link(sample: http://www.youtube.com/watch?v=EujDcc2xEUI) or upload a flv video.
  • Main image. You can select or upload an image in the media gallery or leave this parameter null, so the main image is automatic.
  • Image thumbnail. You can select or upload an image in the media gallery or leave this parameter null, so the image thumbnail is automatic.
  • Title. The title of each video.
  • Description. Description of each video.
  • Subtitles. Subtitles of each video.

 

After the videos, the ultimate has the configuration options and overall design of the ultimate. These are among others: width of the galerie, the galerie height, border, round, % height reserved for the thumbnails into the plugin, number of thumbnails and thumbnails designs.
